# lain

Personal academic website for a CS PhD student. Built as a clean static HTML/CSS/JS site with no frameworks or build steps required.

## Pages

| Page | Description |
|---|---|
| `index.html` | Home / About — profile, news, selected publications |
| `research.html` | Research themes and active projects |
| `publications.html` | Full publication list organized by year |
| `cv.html` | Curriculum vitae |
| `blog/index.html` | Blog post listing |
| `blog/*.html` | Individual blog posts |

## Stack

- Vanilla HTML, CSS (CSS variables, grid, flexbox), and JavaScript — no build tools, no frameworks
- Google Fonts: Inter, Lora, JetBrains Mono
- Fully responsive (mobile/tablet/desktop)

## Getting Started

Just open `index.html` in a browser. No build step required.

To deploy: push to GitHub Pages, Netlify, or any static host.

## Customization

Edit `index.html` to update your name, affiliation, bio, and news.  
Update `css/style.css` CSS variables (`:root` block) to change colors/fonts.  
Add new blog posts by copying an existing post in `blog/` and linking it from `blog/index.html`.
