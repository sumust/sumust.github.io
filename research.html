Research

I'm broadly interested in how we can make computers better thinking partners. My work spans three interconnected themes:

Research Themes

🔬

Human-in-the-Loop Program Synthesis

Building synthesis tools that treat the programmer as a collaborator, not an oracle. I study how partial sketches, interactive feedback, and natural language can guide search in large program spaces.

Program Synthesis PL Theory HCI
🧠

Neurosymbolic Reasoning

Combining the pattern-matching strengths of neural networks with the precision of symbolic AI. I work on hybrid systems for program repair, formal verification, and code generation.

Neurosymbolic AI Formal Methods ML
🤝

Human-AI Interaction in Dev Tools

Empirically studying how developers interact with AI-powered coding assistants — mental models, trust calibration, and workflow integration. I conduct user studies and build prototypes.

User Studies AI Assistants HCI
💬

LLM Reasoning & Prompting

Understanding and improving how large language models handle multi-step compositional reasoning tasks, with a focus on self-consistency, chain-of-thought, and structured prompting.

LLMs Prompting Reasoning

Active Projects

In Progress

SynthChat: A Conversational Interface for Program Synthesis

A VS Code extension that lets developers describe what they want in natural language, incrementally refine the specification through dialogue, and get formally verified code back. Currently running a pilot study with 20 developers.

Synthesis NLP Formal Verification
In Progress

SpecFuzz: Specification-Aware Fuzzing with LLM-Generated Test Oracles

Using LLMs to auto-generate differential test oracles from informal API documentation, then running coverage-guided fuzzing to find spec violations. Target: submission to PLDI 2027.

Testing Fuzzing LLMs
